# หลักสูตรการเขียนโปรแกรม Visual Basic.net
# เนื้อหาของหลักสูตร


1. การเขียนโปรแกรมเบื้องต้น
- ตัวแปร
- ค่าคงที
- ตัวดำเนินการ
- ขอบเขตตัวแปร
2. คอนโทรลเบื้องต้น
- ความหมายของ Property, Method และ Event
- ฟอร์ม
- คอนโทรล
- การใช้งานคอนโทรลเบื้องต้น
3. การควบคุมทิศทางโปรแกรม
- ประโยคคำสั่งแบบเงือนไข IF , ELSE , ELSE IF
- ประโยคคำสั่ง Select Case
- ประโยคคำสั่งแบบวนรอบ Do…Loop , For…Next , While…End While
4. อาร์เรย์
- มิติของอาร์เรย์
- ชนิดของอาร์เรย์
- สแตติกอาร์เรย์
- ไดนามิกอาร์เรย์
5. ฟังก์ชันมาตรฐาน
- ความหมายของฟังก์ชันมาตรฐาน
- ฟังก์ชันการจัดการข้อมูลชนิดตัวเลข
- ฟังก์ชันการจัดการข้อมูลชนิดตัวอักษร
- ฟังก์ชันการจัดการข้อมูลชนิดวันที่
- ฟังก์ชันแปลงค่า
- ฟังก์ชันอื่นๆ
- การเปลียนชนิดของข้อมูลด้วยคลาส Convert
6. โพรซีเยอร์และฟังก์ชัน
- โปรแกรมย่อย
- โพรซีเยอร์
- ฟังก์ชัน
- อาร์กิวเมนต์
- ขอบเขตการใช้งาน
- การใช้งานและคอนโทรลขั้นสูง
7. เมนู
ContextMenuStrip
DataGridView
8. การเขียนโปรแกรมกับฐานข้อมูล
- ฐานข้อมูล
- ภาษา SQL
- การเขียนโปรแกรมกับฐานข้อมูลเบื้องต้น
9. การสร้างฐานข้อมูล SQL Database
- การสร้างฐานข้อมูล
- การสร้างตาราง
- การเขียนโปรแกรมเชื่อมต่อฐานข้อมูล SQL Database
10. การเชื่อมต่อฐานข้อมูล Ms-Access
- การเขียนโปรแกรมเพื่อแสดงผลข้อมูลด้วย Data Reader
- การเขียนโปรแกรมเพื่อแสดงผลข้อมูลด้วย Data Adapter
- การเขียนโปรแกรมเพื่อจัดการข้อมูลในตารางฐานข้อมูล
11. การสร้างแพ็คเก็จ
- การสร้างแพ็คเก็จ
- การติดตั้งแพ็คเก็จ

แนวคิดใหม่ของอิเล็กทรอนิกเฮ้าส์

1. แยกบุคคลที่ทักษะไม่เหมือนกันออกเป็นห้อง โดยคนมีพื้นฐานเรียนอีกห้อง ส่วนคนไม่มีพื้นฐาน และ เรียนช้าอยู่อีกห้อง
2. ให้โอกาศคนที่ไม่มีพื้นฐานได้เรียน โดยจะสอนจนกระทั้งเขาเข้าใจและทำได้ โดยไม่จำกัดเวลาเรียน
3. ปรังปรุงหลักสูตรเรียนใหม่อยู่เสมอเพื่อให้การสอนเป็นไปได้ง่ายขึ้น
4. คนที่เรียนหลักสูตรช่าง จัดฝึกงานหน้าร้านเพื่อฝึกประสบการณ์ก่อนจบหลักสูตร
5. การเรียนมีทั้งภาคภาษาอังกฤษสำหรับชาวต่างชาติ และ ภาคภาษาไทยสำหรับคนไทย
6. ภาคภาษาอังกฤษจะมีการคิดค่าธรรมเนียมเพิ่มเติมเล็กน้อยตามความเหมาะสมของแต่ล่ะหลักสูตร
